# Peer-to-Peer (P2P) Learning Platform

A full-stack web application for collaborative student-led learning.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js (Express)
- **Database:** MongoDB Atlas
- **Auth:** JWT, Bcrypt.js
